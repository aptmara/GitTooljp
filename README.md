# SimplePRClient (GitTooljp)

A simple Git client tool built with WPF and .NET.

## Features

- View repository status
- Manage file changes
- Simple GitHub integration (planned)

## Requirements

- .NET 10.0 (Preview) or later
- Windows OS

## Getting Started

1. Clone the repository.
2. Open the solution in Visual Studio or your preferred IDE.
3. Build and run the project.
